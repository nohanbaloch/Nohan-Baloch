# Advanced PDF Reader

This project provides a comprehensive solution for viewing and manipulating PDF documents. Originally designed as a standalone tool, it has been optimized for performance in diverse environments.

## Key Features

- **High-Performance Rendering**: Utilizing specialized rendering engines for fast page loads.
- **Page Navigation**: Intuitive controls for zooming, searching, and rotating pages.
- **Document Management**: Capabilities for merging, splitting, and metadata extraction.
- **System Integration**: Seamlessly hooks into the parent OS for file operations.

## Tech Stack

- **Core**: Go (Golang)
- **UI Framework**: Fyne
- **Libraries**: pdfcpu, go-fitz
- **Processing**: Optimized CGO bindings for speed
