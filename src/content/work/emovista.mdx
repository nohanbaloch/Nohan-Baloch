# EMOVISTA

EMOVISTA leverages state-of-the-art deep learning models to perform real-time facial expression recognition and sentiment analysis.

## Tech Stack

- ML Core: Python, TensorFlow/Keras
- Frontend: CustomTkinter / Web Interface
- Processing: OpenCV for real-time video streaming

## Features

- Real-time emotion detection
- High accuracy classification
- Low-latency processing
