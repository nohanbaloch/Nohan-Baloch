# UniSchedule

UniSchedule is a specialized tool designed to solve the complexity of academic scheduling.

## Key Features

- Interactive timetable generation
- Multi-theme support (Onyx, Alabaster, Spectrum)
- Automated conflict detection
- PWA support for offline access
