# Sensor-Web Project

Sensor-Web provides a robust infrastructure for monitoring distributed sensor networks via a centralized web dashboard.

## Technical Details

- Real-time data streaming via WebSockets.
- Dynamic charting and health monitoring.
- Scalable backend for concurrent node connections.
