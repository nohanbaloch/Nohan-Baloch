# Astronomical Prediction and Classification System

This project focuses on the classification of celestial bodies using high-dimensional astronomical data.

## Highlights

- Multi-class classification of stars, galaxies, and quasars.
- Advanced data preprocessing for noisy sensor data.
- Integration of physical features into machine learning models.
